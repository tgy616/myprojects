==================== 变量表达式(Variable Expressions) ===============================
${user.name} -->
'The name of the user is ' + ${user.name} -->

=====================消息表达式：Message Expressions =================================
#{home.welcome} --> Welcome to our grocery store!
#{home.welcome.replace(${user.name})} -->

========================== Literals =======================================
Now you are looking at a template file.
仅仅输出数字 th:text="2013" --> 1492.
数字计算 th:text="2013 + 2" --> 1494.
${user.isAdmin()} == false --> false
${user.other} == null --> null
========================== 文本操作符： Text operations ==================================
'The name of the user is ' + ${user.name} + '_' + ${user.age} -->
|Welcome to our application, ${user.name}!| -->
等价于这条语句：
'Welcome to our application, ' + ${user.name} + '!' -->
========================== 算术表达式：Arithmetic operations ==================================
${user.age} % 2 == 0 -->
true -->
!(${user.age} % 2 == 0 -->
(${user.age} % 2 == 0) and true -->
========================== 比较操作符：Comparisons and equality ==================================
${user.age} > 18 --> 大人
${user.age} != 18 --> 大人_no_equality

========================== 条件操作符： Conditional operators ==================================
${user.age}%2==0 ? 'even' -->
${user.age}%2==0 ? 'even' : 'odd' -->
${user.age} ?:'18' -->
